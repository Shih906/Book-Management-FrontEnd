## 簡介
此專案為自主練習項目，搭配 .NetCore WebApi 翻新舊有 Asp.Net MVC 圖書管理系統基本功能，WebApi 專案可參考[此處](https://github.com/Shih906/Book-Management-WebApi)

## 安裝及啟動
* #### 安裝Node.js npm　:　https://nodejs.org/en/
* #### 安裝 pnpm : npm install pnpm -g
* #### 啟動Server　:　`pnpm dev`
